Добро пожаловать!

В данной вики собрана вся полезная информация, которую вам нужно знать для комфортной игры на планшетах Microsoft Surface либо на любом другом, основанном на процессоре Intel Atom.

Прежде всего нужно рассказать про основы. Лучше всего установить Windows 8.1, нежели пользоваться Windows 10 или какой-нибудь сборкой Windows 11. У этого решения есть как плюсы, так и минусы. Из плюсов: низкое потребление оперативной памяти, быстрая скорость загрузки ОС и её быстрая работа в целом.

Скачать её можно с официального сайта Microsoft: Загрузить Windows 8.1.

А вот из минусов это несовместимость с современными версиями программ. Например:

  • Adobe Photoshop
  • Visual Studio Code
  • Steam
  • Любые браузеры на основе Chromium
и другие...

На данный момент Steam всё ещё работает, но с 01.01.2024 он прекратит поддержку Windows 8.1. Если вас это не пугает и вам не нужны самые свежие версии различных программ, то Windows 8.1 - это отличный выбор. Однако, я предлагаю вам установить Windows 8.1 в качестве основной системы и Windows 10 / Windows 11 (по выбору) в качестве дополнительной, как раз на такой случай, если какая-либо программма не поддерживает Windows 8.1. Сделать это можно по инструкции на этом сайте.
